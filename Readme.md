# StreamFlix Analytics

## Requisitos previos
Antes de ejecutar el proyecto, coloca los siguientes archivos en `src/main/resources/data/`:
- `server_logs.txt` → logs de servidor para el Módulo 1
- `movies_metadata.csv` → catálogo de películas para el Módulo 2

Estos archivos no están incluidos en el repositorio por su tamaño.

## Ejecución
Usar las configuraciones de IntelliJ:
- **Modulo 1** → argumento `1`
- **Modulo 2** → argumento `2`
